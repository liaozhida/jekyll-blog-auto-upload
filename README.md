## 博客站点

#### segmentfault
- 支持markdown格式的文章提交
- 一天提交不能超过30篇文章，超过之后需要填写验证码

#### segmentfault
- 支持html格式的文章提交
- 一天提交不能超过10篇文章，超过之后需要填写验证码
